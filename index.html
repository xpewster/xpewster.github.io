MISSILECAVE

GM:StudioGML

Missilecave is a challenging 2d platformer game set in a mysterious cave, drawing inspiration from the 2010 hit Super Meat Boy. It features numerous levels, mechanics, and a complete UI. This project was started while I was in high school and I designed and coded all the gameplay. I also drew all the art. Missilecave is made in GameMaker:Studio using the GML scripting language.

TERRAINGEN

Java

A heightmap generator that can be used to create realistic terrain maps. The app implements a modified diamond-square algorithm and then processes the data for rendering or export. It is made in Java with Swing for the UI.
